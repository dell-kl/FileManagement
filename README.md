# SISTEMA ADMINISTRACIÓN DE ARCHIVOS
[Industria Aeronaútica del Ecuador](https://www.diaf.gob.ec/)

---

## Propósito del Sistema

El sistema fue planeado con el fin de gestionar documentos internos de la empresa, que en su 
lenguaje técnico lo denominaban como régimen.

Cada departamento era encargado de generar un documento (que es un proceso) de una compra que 
se ha realizado. Para no tener solamente un respaldo físico de los documentos, se creó este
sistema para que cada departamento pueda subir de manera virtualizada cada archivo. Por ende
el sistema se divide por perfiles y permisos.
